# Cart Total Amount Calculator

A simple web application that calculates and displays the total amount of items in a shopping cart.

## Features
- Real-time cart total calculation
- Clean and responsive design
- Easy to extend with more cart functionality

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
cart-total-amount/
├── index.html       # Main HTML file
├── styles.css       # Styling
└── script.js        # Cart calculation logic

## How to Run
1. Clone or download this repository
2. Open `index.html` in any modern web browser
   - Double-click the file, or
   - Use a local server (recommended):
